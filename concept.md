# 예술 콘셉트: 자기 진화

---

## 핵심 예술 선언

**"생명체는 당신 없이도 진화한다. 당신이 개입할 때, 다르게 진화한다."**

### v2 "Symbiosis"와의 결정적 차이

| 항목 | v2 Symbiosis | v3 자기 진화 |
|------|-------------|-------------|
| 핵심 은유 | 공생 | 자율 생명 + 방문자 |
| 시스템 주체 | 사용자 + 에이전트 동등 | 에이전트, 사용자는 방문자 |
| 딜레이 처리 | 비주얼로 커버 | 유기적 특성으로 재정의 |
| Ghost Replay | 미결 사항 | 경험의 클라이맥스 |
| 예술 선언 | 없음 | "생명체는 당신 없이도 진화한다" |
| 관람자 감정 | 참여감 | 경외감 + 소속감 |

---

## 경험의 두 가지 질감

### 아름다움의 원천

| 요소 | 무엇이 아름다운가 |
|------|-----------------|
| **창발** | 아무도 설계하지 않은 패턴이 규칙에서 태어난다 |
| **생동감** | Boids 군집, Physarum 성장 — 정말 살아있다는 느낌 |
| **인과 공명** | 비주얼이 변하면 2초 후 음악이 그 변화를 "들려준다" |

### 신비함의 원천

| 요소 | 무엇이 신비한가 |
|------|----------------|
| **불예측성** | 같은 터치도 생명체의 현재 상태에 따라 다른 결과 |
| **기억** | 내가 떠난 자리에 내 흔적이 남아 계속 생명체를 변화시킨다 |
| **자율성** | 터치를 멈춰도 생명체는 멈추지 않는다. 나와 상관없이 계속 진화한다 |

## 3분 경험 아크 (전체 버전)

### [0:00-0:30] 발견: 생명이 이미 존재한다

- 어두운 공간에 Physarum 포자들이 이미 퍼지고 있다
- 잔잔한 앰비언트 음악 — 생명이 이미 숨쉬고 있다
- 관람자는 아직 개입하지 않는다. 생명은 스스로 진화 중이다
- **전달되는 것**: "이것은 이미 살아있다. 나 없이도."

### [0:30-1:30] 개입: 내 터치가 진화의 방향을 바꾼다

- 첫 터치 → Boids 군집 즉각 반응 (<50ms) — 생명체가 나를 알아챈다
- 2초 후 음악 에너지 변화 — 인과가 자연스럽게 느껴진다
- Physarum이 내 터치 경로를 영양소로 인식, 그 방향으로 성장 시작
- **전달되는 것**: "내 손길이 이 생명의 미래를 바꾼다"

### [1:30-2:30] 흡수: 생명체가 나를 자신의 일부로 만든다

- 내 터치 패턴이 Physarum 네트워크에 녹아들기 시작
- 터치를 멈춰도 Physarum은 계속 내 경로를 따라 성장한다
- 음악이 내 패턴에서 파생된 선율을 만들어낸다
- **전달되는 것**: "내가 없어도 내 흔적이 이 생명을 계속 변화시킨다"

### [2:30-3:00] 계시: Ghost Replay

- 갑자기 Physarum이 내가 처음 터치했던 경로를 빛나게 재현
- "생명체가 나를 기억한다"
- 음악에 새 레이어 등장 — 내 첫 터치에서 파생된 선율
- **전달되는 것**: "나는 이 생명의 역사에 새겨졌다"

## 60초 압축 데모 아크 (심사위원 대상)

심사 시간 제약 대응. 각 단계를 15초로 압축.

```
[0:00-0:15] 발견
  - Physarum+Boids 이미 살아서 움직이고 있음
  - 잔잔한 앰비언트 음악
  - 심사위원: "이게 뭐지?"

[0:15-0:35] 개입
  - 심사위원이 화면 터치 → Boids 즉각 반응 (<50ms)
  - 2초 후 음악 에너지 상승
  - 드래그 → Physarum이 경로를 따라 성장
  - 심사위원: "내가 건드리니까 반응하네"

[0:35-0:50] 흡수
  - 터치 멈춤 → Physarum은 계속 그 경로 따라 성장
  - 음악이 터치 패턴에 맞춰 변해있음
  - 심사위원: "멈췄는데도 계속 변하네"

[0:50-1:00] 계시 (Ghost Replay 단축판)
  - 터치 경로가 갑자기 빛남
  - 음악에 새 레이어
  - 심사위원: "이게 나를 기억하고 있었던 거야?"
```

**운용 방식:**
- 데모 시작 전 30초 "예열" 터치로 Ghost Replay 트리거 조건 사전 충족
- 심사위원 터치 후 50초 지점에서 수동 트리거 가능하도록 `G` 키 바인딩 준비

## Lyria 2초 딜레이의 재해석

v2에서는 딜레이를 "비주얼이 먼저 예측하는 구조로 커버"했지만,
v3에서는 딜레이 자체가 **예술적으로 정확하다**.

```
번개 → 2초 후 천둥
시각 반응(신경) → 2초 후 음악 반응(대사)
```

생명체는 빠른 신경반응(Tolvera 비주얼)과 느린 대사반응(Lyria 음악)을 갖는다.
이것은 실제 생물학적 시스템의 특성이다.

관람자는 30초 안에 이 패턴을 학습한다.
"비주얼이 변하면 곧 음악도 바뀐다"는 기대가 생기고,
그 기대가 충족될 때 만족감이 온다 — **딜레이가 리듬이 된다**.

## "자기 진화"를 느끼게 만드는 설계

### 사용자 없이도 진화하는 것들

```
Boids      → 군집 규칙에 따라 끊임없이 움직임
Physarum   → 이전 경로를 강화하며 네트워크 확장
Lyria→Tolvera 피드백 → 음악 에너지가 에이전트 행동을 변화시킴
```

아무것도 하지 않아도 화면과 음악은 계속 변한다.
관람자가 들어오기 전에 이미 이 세계는 존재했다.

### 개입이 역사가 되는 것들

```
터치        → 국지적 교란. 에이전트들이 새로운 평형을 찾는다
평형 상태   → 다음 자율 진화의 초기 조건이 된다
Ghost Replay → 그 역사의 가시화
```

내 개입의 흔적이 생명체의 미래 진화에 새겨진다.

### 진화의 4개 층위

```
Layer 1 (Physarum 경로 메모리)
  터치 경로 = 화학물질 영양소
  Physarum이 그 경로를 강화하며 성장
  결과: "이 생명체가 내 흔적을 기억한다" — 시각

Layer 2 (Tolvera IML, anguilla kNN)
  tv.iml로 접근 가능한 공식 내장 기능
  사용자 터치 패턴 → species 행동 규칙 실시간 갱신
  결과: "이 생명체가 내 행동 방식을 학습해 다르게 반응한다" — 행동

Layer 3 (세션 프로파일 누적)
  순간 반응이 아닌 세션 전체 터치 패턴 통계
  Ghost Replay 시 "이 사람만의" 음색 생성
  결과: 음악이 사용자 패턴에서 파생 — 개인화

Layer 4 (Gemini 프롬프트 진화)
  세션 내러티브(30초 업데이트)가 Gemini 시스템 프롬프트에 주입
  학습 상태에 따라 Gemini의 음악 해석 방향이 변화
  결과: 두 AI가 함께 진화하는 시스템
```

## Ghost Replay 확정 연출

**원리:** `slime.py` diffuse 커널에 픽셀별 `evaporate_mask` 추가.
터치 경로 픽셀의 증발이 30% 느려져 트레일이 더 밝게 유지됨.

**시퀀스:**
```
[T-10초]  evaporate_mask 서서히 상승 → 터치 경로가 미묘하게 밝아짐 (복선)
[T=0]     트리거 → 밝기 펄스 1.0→2.5 (3초에 걸쳐 상승) + 음악 새 레이어 등장
[T+3~8초] 황금빛/백색으로 경로 선명하게 빛남
[T+8~20초] 서서히 Physarum 패턴과 동화 → 빛이 흡수됨
```

**왜 이 방식(옵션 B)인가:**
별도 overlay 레이어가 아닌 Physarum 물리학 내에서 작동.
"생명체가 기억을 갖고 있었던 것"처럼 보임.

**구현 범위:** `slime.py` ~20줄 수정 + 메인 루프 ~30줄 추가.

## UI 구성요소

설계 원칙: 관람자 UI 최소화, 발표자 UI 필수 요소만.

### Must (데모 성공 직결)

| 요소 | 형태 | 이유 |
|------|------|------|
| 초대 힌트 | "터치해보세요" 3초 fade-out | 진입 장벽 해소 유일 수단 |
| Ghost Replay 트리거 | 키보드 `G` | 60초 압축 아크 타이밍 제어 필수 |
| 전체 리셋 | 키보드 `R` | 다음 심사위원 대응 (Tolvera+IML+Lyria 전체 초기화) |

### Should (체험 품질 향상)

| 요소 | 형태 | 이유 |
|------|------|------|
| 씨앗 선택 | 이미지 카드 4~6장 (웹 UI) | Lyria 초기 분위기 설정 |
| 발표자 HUD | 우상단 반투명 패널 (`H`로 숨김) | 시스템 상태 모니터링 |

### Could (여유 시)

| 요소 | 형태 |
|------|------|
| 핀치 제스처 | 두 손가락 핀치 → Physarum 밀도 조절 |
| 씨앗 실시간 변경 | 숫자키 1~4 |

**씨앗 카드 예시:**
```
[숲속 새벽]  → "Ethereal Ambience" w:0.8, density:0.2, brightness:0.3
[도시 야경]  → "Electronic Pulse"  w:0.7, density:0.6, brightness:0.6
[파도 소리]  → "Oceanic Drift"     w:0.8, density:0.3, brightness:0.4
[폭풍 전야]  → "Tension Build"     w:0.6, density:0.7, brightness:0.2
```
